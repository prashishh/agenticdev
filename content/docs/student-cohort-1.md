I'm selecting 5-10 students for a one-month guided program to help them transition from traditional coding to agentic coding. This is a hands-on cohort where you'll work directly with me, learning how to use coding agents effectively across real projects. You should have solid experience and knowledge in coding, have worked on several projects, and ideally be a 3rd or 4th year student with internship experience. 

This is not intended for working professionals or independent practitioners. The program is completely free, and you'll get access to coding agents for the duration of the cohort.
